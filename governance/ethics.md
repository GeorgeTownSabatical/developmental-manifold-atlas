# Ethics and Privacy

Developmental Manifold Atlas is designed to coordinate research without becoming a dangerous data dump.

## Public Repository Allows

- synthetic records
- public animal datasets
- aggregate human summaries
- dataset metadata
- schema definitions
- validation tools
- analysis scripts
- reproducibility notebooks
- dashboard-ready summaries

## Public Repository Prohibits

- identifiable fertility records
- raw neonatal clinical records
- identifiable genetic data
- protected health information
- uncontrolled participant-level human data
- direct identifiers such as names, emails, phone numbers, addresses, medical record numbers, or dates of birth

## Data Tiers

| Tier | Data type |
| --- | --- |
| Tier 0 | Synthetic or demo data |
| Tier 1 | Public animal data |
| Tier 2 | De-identified aggregate human data |
| Tier 3 | Controlled-access human clinical data |
| Tier 4 | Sensitive genetics, neonatal, or fertility data represented only by metadata |
