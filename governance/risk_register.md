# Risk Register

| Risk | Mitigation |
| --- | --- |
| Scientific overclaiming | strict evidence grades and uncertainty |
| Privacy harm | no sensitive individual human data in public repo |
| Misuse | ethics policy, anti-discrimination language, controlled access |
| Data inconsistency | schemas, quality flags, landmark equivalence classes |
| Contributor burnout | modular tasks, working groups, clear roadmap |
| Institutional hesitation | aggregate-first templates and ethics-first messaging |
| Correlation mining | pre-registration, null models, replication rules |
