# Phase 2 Standardization Checklist

Use this checklist before proposing a dataset or measurement protocol.

- Dataset has version, provenance, limitations, access tier, and update frequency.
- Organism or aggregate context is recorded.
- Measurement record has trait ID, value, unit, landmarks, method, instrument, observer, uncertainty, quality flag, source, access tier, and provenance.
- Landmark IDs map to `registry/landmarks.json`.
- Cross-species measurements state an equivalence class from `registry/equivalence_classes.json`.
- Derived variables use formulas from `registry/derived_variables.json`.
- Protocol readiness is represented in `registry/protocol_readiness.json`.
- Sensitive human data is not placed in the public repository.
