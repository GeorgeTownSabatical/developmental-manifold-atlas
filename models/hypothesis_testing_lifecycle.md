# Hypothesis Testing Lifecycle

1. Specify claim.
2. Define required variables.
3. Define inclusion criteria.
4. Define exclusion criteria.
5. Define correction factors.
6. Run baseline analysis.
7. Run null models.
8. Run sensitivity analysis.
9. Run replication analysis when possible.
10. Assign evidence grade.
11. Update dashboard outputs.

Evidence grades:

| Grade | Meaning |
| --- | --- |
| E0 | Conceptual |
| E1 | Pilot signal |
| E2 | Single-dataset support |
| E3 | Multi-dataset support |
| E4 | Replicated |
| E5 | Strong |
| C | Contradicted |
