Data Standards

No Naked Measurements

Every value must carry anatomy, biological context, provenance, units, quality, ethics, and reproducibility metadata.

Universal Measurement Record

Measurement records link organism context, dataset provenance, trait vocabulary, exact landmarks, method, instrument, observer, uncertainty, quality flag, source, and access tier.

Quality Grades

A
Exact standard landmark, trained observer, repeated measurement.
B
Exact landmark, single measurement.
C
Comparable but alternate landmark.
D
Approximate or proxy landmark.
F
Unusable or ambiguous.

Missingness Codes

Use explicit missingness: NA_NOT_MEASURED, NA_NOT_REPORTED, NA_NOT_APPLICABLE, NA_RESTRICTED, or NA_UNKNOWN.
