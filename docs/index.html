Developmental Manifold Atlas

A public, forkable research platform for mapping reproductive anatomy, craniofacial geometry, neonatal growth, fertility outcomes, genetics, endocrine exposure, and environment into a shared developmental state-space.

Developmental State Objects

Every record maps observations into a common object with identity metadata, anatomy, physiology, exposure, genetics, developmental history, reproductive outcomes, and provenance.

Public-Safe by Design

The public repository stores schemas, metadata, synthetic records, open animal data, aggregate summaries, validation tools, and dashboards, not identifiable clinical data.

Versioned Hypotheses

Claims become testable records with required variables, correction factors, eligible datasets, replication thresholds, and current evidence status.
