## Phase 2 Standardization Rules

- Every measurement must cite a landmark start and endpoint.
- Every cross-species comparison must state an equivalence class.
- Every derived value must preserve the raw input values.
- Every protocol must define minimum fields and review status.
- Human neonatal, fertility, and genetic measurements remain aggregate, synthetic, metadata-only, or controlled-access.
