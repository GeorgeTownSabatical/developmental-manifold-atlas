# Neonatal Measurement Protocol v0.2

## Core Measures

| Measure | Definition |
| --- | --- |
| Birthweight | Standardized neonatal weight |
| Crown-heel length | Total neonatal length |
| Head circumference | Occipitofrontal circumference |
| AGD | Sex-specific neonatal AGD |
| Gestational age | Weeks and days |
| Placental weight | Developmental resource proxy |
| Apgar | Immediate neonatal stability |

Only aggregate, synthetic, or controlled-access metadata belongs in the public repository.

## Required Context

- gestational age
- timing after birth
- birth context
- sex
- clinical measurement source
- ethics status
- access tier
- aggregation level

AGD in neonates must be performed only under approved clinical or research protocols. Public dashboards should use synthetic examples, metadata-only pointers, or aggregate summaries.
