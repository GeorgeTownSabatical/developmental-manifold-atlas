# AGD Measurement Protocol v0.2

## Scope

This protocol defines placeholder landmarks for anogenital distance measurements across public-safe demonstration workflows. Real human, veterinary, or animal-study protocols must be approved by the appropriate ethics, clinical, or animal-care body before use.

## Male Measures

| Measure | Definition |
| --- | --- |
| AGD-AS | Anus to posterior scrotal base |
| AGD-AP | Anus to base of penis |
| Scrotal width | Transverse scrotal measure |
| Perineal body length | Anus to genital midpoint |

## Female Measures

| Measure | Definition |
| --- | --- |
| AGD-AC | Anus to clitoris |
| AGD-AF | Anus to posterior fourchette |
| Perineal body length | Anus to genital midpoint |

## Required Metadata

- species
- sex
- age or developmental stage
- body size measure
- instrument
- measurement posture
- landmark_start
- landmark_end
- equivalence_class
- landmark confidence
- repeated measurement count
- intra-observer error
- inter-observer error when available
- observer or lab code
- ethics status
- access tier

## Positioning

Record subject position, restraint or handling condition, temperature or anesthesia context when relevant, and whether body mass and body length were recorded on the same day.

## Exclusion Criteria

- ambiguous start or endpoint landmark
- missing unit
- unknown sex when sex-specific landmark interpretation is required
- uncontrolled human individual data
- protocol deviations without notes
