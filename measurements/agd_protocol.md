# AGD Measurement Protocol v0.1

## Scope

This protocol defines placeholder landmarks for anogenital distance measurements across public-safe demonstration workflows. Real human, veterinary, or animal-study protocols must be approved by the appropriate ethics, clinical, or animal-care body before use.

## Male Measures

| Measure | Definition |
| --- | --- |
| AGD-AS | Anus to posterior scrotal base |
| AGD-AP | Anus to base of penis |
| Scrotal width | Transverse scrotal measure |
| Perineal body length | Anus to genital midpoint |

## Female Measures

| Measure | Definition |
| --- | --- |
| AGD-AC | Anus to clitoris |
| AGD-AF | Anus to posterior fourchette |
| Perineal body length | Anus to genital midpoint |

## Required Metadata

- species
- sex
- age or developmental stage
- body size measure
- instrument
- measurement posture
- landmark confidence
- observer or lab code
- ethics status
- access tier
