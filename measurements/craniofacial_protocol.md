# Craniofacial Measurement Protocol v0.1

## Core Measures

| Measure | Definition |
| --- | --- |
| Skull length | Cranial base to rostrum |
| Snout length | Nasion to rostral tip |
| Bizygomatic width | Cheekbone width |
| Facial height | Upper face vertical dimension |
| Mandible length | Condyle to symphysis |
| Cranial index | Skull width divided by skull length |
| Orbit spacing | Interorbital distance |
| Jaw angle | Mandibular angle metric |

## Derived Ratios

- `snout_length / skull_length`
- `skull_width / skull_length`
- `facial_width / upper_facial_height`
