# Craniofacial Measurement Protocol v0.2

## Core Measures

| Measure | Definition |
| --- | --- |
| Skull length | Cranial base to rostrum |
| Snout length | Nasion to rostral tip |
| Bizygomatic width | Cheekbone width |
| Facial height | Upper face vertical dimension |
| Mandible length | Condyle to symphysis |
| Cranial index | Skull width divided by skull length |
| Orbit spacing | Interorbital distance |
| Jaw angle | Mandibular angle metric |

## Derived Ratios

- `snout_length / skull_length`
- `skull_width / skull_length`
- `facial_width / upper_facial_height`

## Image-Based Metadata

- camera or scanner model
- scale marker
- angle or view
- lighting
- landmark set
- software used
- observer
- repeatability score
- equivalence class for species-specific landmarks

Human fWHR is not directly equivalent to dog muzzle width. Comparative analyses must state whether landmarks are direct, biologically comparable, alternate, proxy, or unusable.
